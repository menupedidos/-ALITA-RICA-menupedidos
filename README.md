# 🍔🔥 ALITA RICA - SABOR DE BARRIO AL CARBÓN

Sistema de menú digital interactivo con envío automático de pedidos por WhatsApp.

---

## 🚀 FUNCIONES DEL SISTEMA

✔ Campo obligatorio del nombre del cliente  
✔ Opción Pickup o Delivery  
✔ Dirección obligatoria si elige Delivery  
✔ Selección de productos por cantidad  
✔ Selección de sabores para Alitas y Boneless  
✔ Selección de forma de pago (Efectivo o Transferencia)  
✔ Cálculo automático del total  
✔ Generación automática del mensaje listo para WhatsApp

# ⚙️ ¿CÓMO FUNCIONA?

1. El cliente ingresa su nombre.
2. Selecciona Pickup o Delivery.
3. Si elige Delivery debe ingresar dirección obligatoria.
4. Selecciona cantidades de productos.
5. Selecciona sabores para Alitas y Boneless.
6. Selecciona forma de pago.
7. El sistema calcula automáticamente el total.
8. Se genera un mensaje listo para enviar por WhatsApp.

---

# 📲 IMPLEMENTACIÓN

1. Subir `index.html` al repositorio.
2. Activar GitHub Pages.
3. Compartir el link generado.
4. Los clientes podrán hacer pedidos desde cualquier celular.
