# 🔥 ALITA RICA - Menú Digital

Menú digital interactivo para pedidos vía WhatsApp.

## 🍔 Productos
- Burgers
- Wings
- Boneless
- Jochos

## 📲 Cómo funciona
1. El cliente selecciona productos.
2. Se calcula el total automáticamente.
3. El pedido se envía directo a WhatsApp.

## 🚀 Instalación como App
Este menú funciona como PWA (App instalable).
