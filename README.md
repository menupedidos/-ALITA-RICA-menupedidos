# 🔥 ALITA RICA 🔥 – Sabor de Barrio al Carbón 🍗🍔

Menú digital interactivo para pedidos en línea con envío automático a WhatsApp.

---

## 📌 Descripción

Proyecto desarrollado en **HTML5, CSS3 y JavaScript** que permite a los clientes realizar pedidos digitales desde su celular y enviarlos automáticamente a WhatsApp.

El sistema incluye:

✅ Visualización completa del menú  
✅ Cálculo automático del total en tiempo real  
✅ Selección obligatoria de exactamente 3 sabores para Alitas  
✅ Selección obligatoria de exactamente 3 sabores para Boneless  
✅ Campo para indicar cuántas piezas se bañan en cada sabor seleccionado  
✅ Campo obligatorio de dirección para pedidos Delivery  
✅ Selección de forma de pago (Efectivo o Transferencia)  
✅ Generación automática del pedido detallado con cantidades  
✅ Experiencia optimizada para celular  
✅ Envío directo a WhatsApp con desglose completo del pedido

## 🚀 Cómo usar este proyecto

1. Clona el repositorio
2. Abre el archivo `index.html`
3. Edita el número de WhatsApp dentro del código JavaScript: let telefono = "5219903667103";
4. Guarda los cambios
5. Sube el repositorio a GitHub
6. Activa GitHub Pages:
- Settings → Pages  
- Branch: main  
- Folder: /(root)
7. Comparte tu enlace público
