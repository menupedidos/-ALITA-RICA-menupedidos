
# 🔥 ALITA RICA – Sabor de Barrio al Carbón 🍗🍔

Menú digital interactivo para pedidos en línea con envío automático a WhatsApp.

---

## 📌 Descripción

Este proyecto es un menú digital desarrollado en HTML, CSS y JavaScript que permite a los clientes:

✅ Ver el menú completo  
✅ Elegir hasta 3 sabores para Alitas y Boneless  
✅ Ingresar dirección obligatoria para Delivery  
✅ Enviar pedido directo a WhatsApp  
✅ Experiencia optimizada para celular  

---

- Doble bacón c/ papas

## 🚀 Cómo usar este proyecto

1. Clona el repositorio
2. Edita el número de WhatsApp en `index.html`
3. Sube los cambios
4. Activa GitHub Pages en:
   Settings → Pages → Branch: main → Folder: /(root)
5. Comparte tu link público
