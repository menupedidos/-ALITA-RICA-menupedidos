## 📌 Descripción

Este proyecto es un menú digital desarrollado en HTML, CSS y JavaScript que permite a los clientes:

- Ver el menú completo
- Seleccionar cantidades por producto
- Elegir exactamente 3 sabores para Alitas
- Elegir exactamente 3 sabores para Boneless
- Seleccionar Pickup o Delivery
- Ingresar dirección obligatoria si elige Delivery
- Seleccionar forma de pago (Efectivo o Transferencia)
- Visualizar el total en tiempo real
- Enviar el pedido automáticamente por WhatsApp
