# 🔥 ALITA RICA  
## SABOR DE BARRIO AL CARBÓN

Sistema de pedidos en línea con cálculo automático y envío directo a WhatsApp.

---

## 📖 Sobre el Proyecto

Este sistema permite a los clientes realizar su pedido de manera rápida desde cualquier dispositivo.  
El total se calcula automáticamente y el pedido se envía listo por WhatsApp.

No requiere base de datos ni servidor.  
Funciona directamente con GitHub Pages.

## 🚀 Cómo Publicarlo en GitHub Pages

1. Crear un repositorio nuevo.
2. Subir el archivo `index.html`.
3. Ir a **Settings → Pages**.
4. Seleccionar rama `main`.
5. Guardar cambios.
6. Usar el enlace generado para compartir tu menú.
