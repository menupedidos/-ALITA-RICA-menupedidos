# 🍔🔥 ALITA RICA – SABOR DE BARRIO AL CARBÓN

Sistema de menú digital interactivo con generación automática de pedido para WhatsApp.

Este proyecto permite que el cliente:

✅ Seleccione productos del menú  
✅ Ingrese su nombre (obligatorio)  
✅ Elija Pickup o Delivery  
✅ Agregue dirección obligatoria si selecciona Delivery  
✅ Seleccione hasta 3 sabores para Wings y Boneless  
✅ Elija forma de pago (Efectivo o Transferencia)  
✅ Envíe su pedido automáticamente por WhatsApp

## ⚙️ FUNCIONALIDADES DEL SISTEMA

- Campo obligatorio para nombre del cliente
- Selección dinámica de productos
- Control de hasta 3 sabores para alitas y boneless
- Validación automática de dirección en Delivery
- Selección de método de pago
- Generación automática de mensaje formateado
- Enlace directo a WhatsApp con pedido listo para enviar
- Diseño limpio, funcional y adaptable a celular

---

## 📲 Cómo usar

1. Abrir el archivo `index.html`
2. Seleccionar productos
3. Completar datos obligatorios
4. Presionar **Enviar Pedido**
5. Se abrirá WhatsApp con el pedido listo para confirmar

## 🚀 Publicación en GitHub Pages

1. Subir archivos al repositorio
2. Ir a **Settings**
3. Entrar en **Pages**
4. Seleccionar rama `main`
5. Guardar
6. Listo 🎉 tu menú estará en línea
