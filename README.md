# 🔥 ALITA RICA – Sabor de Barrio al Carbón 🍔🍗

Menú digital interactivo con cálculo automático y envío directo de pedidos a WhatsApp.

---

## 📌 Descripción del Proyecto

Este sistema fue desarrollado en **HTML, CSS y JavaScript** para permitir a los clientes realizar pedidos en línea de manera rápida y sencilla.

El pedido se genera automáticamente con el nombre del cliente, cantidades seleccionadas, sabores elegidos, forma de pago y total calculado en tiempo real.

No requiere base de datos ni servidor.  
Funciona directamente con GitHub Pages.

---

## ✅ Funcionalidades Incluidas

- Nombre del cliente obligatorio  
- Opción de Pickup o Delivery  
- Dirección obligatoria si selecciona Delivery  
- Selección de cantidad por producto  
- Selección EXACTA de 3 sabores para Alitas  
- Selección EXACTA de 3 sabores para Boneless  
- Forma de pago (Efectivo o Transferencia)  
- Visualización del total en tiempo real  
- Generación automática del mensaje para WhatsApp  
- Diseño optimizado para celular

## 🚀 Cómo Publicar el Proyecto

1. Crear un nuevo repositorio en GitHub.
2. Subir el archivo `index.html`.
3. Ir a **Settings → Pages**.
4. En “Source” seleccionar la rama `main`.
5. Guardar cambios.
6. Compartir el enlace público generado.
